# SurnameGussr
In the same vein as other Gussr-type games. This is Surname Gussr, 5 rounds each with a different surname from around the world and you have to guess where it is from.

#Daily (Once per Day)
Choose Daily to go Through 5 rounds that change everyday, With the points from this mode ranking you on a Global Leaderboard 

#Play(Unlimited Play)
Same amount of rounds as the Daily mode, but instead there is no end and it will endlessly generate 5 rounds for you to guess
